# 🧪 줄넘기 경기 매니저 테스트 체크리스트

**테스트 날짜:** 2025-11-24
**개발 서버:** http://localhost:3001
**테스트 대상:** 종목 선택 자동 모달 + 전체 선택 버튼 + 실시간 학급 동기화

---

## 📋 사전 준비

### ✅ 1. 초기 환경 설정
- [ ] 브라우저에서 http://localhost:3001 접속
- [ ] 로그인 완료 (Google 계정)
- [ ] 개인정보 동의 완료
- [ ] 학년 선택 (예: 5학년)

### ✅ 2. 기본 데이터 준비
- [ ] 학급 최소 2개 이상 등록 (예: 5-1반, 5-2반)
- [ ] 각 학급에 학생 5명 이상 등록
- [ ] 종목은 기본 종목 사용 (개인/짝/단체 각 1개 이상)

---

## 🎯 테스트 시나리오

### 테스트 1️⃣: 실시간 학급 동기화 (학급 추가)

**목적:** 학급 관리 모달 열린 상태에서 학급 추가 시 즉시 반영 확인

**단계:**
1. [ ] "학급 관리" 버튼 클릭하여 모달 열기
2. [ ] 현재 표시된 학급 개수 확인 (예: 2개 학급)
3. [ ] 모달을 **닫지 않고** "새 학급 추가" 버튼 클릭
4. [ ] 학급 생성 모달에서 학급명 입력 (예: 5-3반)
5. [ ] 학생 5명 추가
6. [ ] "학급 생성" 버튼 클릭

**예상 결과:**
- [ ] ✅ 학급 관리 모달이 **자동으로 업데이트**되어 새 학급(5-3반) 표시
- [ ] ✅ 모달 하단 통계: "전체 3개 학급, XX명의 학생" 즉시 업데이트
- [ ] ✅ 새로고침 없이 실시간 반영

**실패 조건:**
- [ ] ❌ 모달을 닫고 다시 열어야만 새 학급이 보임

---

### 테스트 2️⃣: 실시간 학급 동기화 (학급 삭제)

**목적:** 학급 관리 모달 열린 상태에서 학급 삭제 시 즉시 반영 확인

**단계:**
1. [ ] "학급 관리" 버튼 클릭하여 모달 열기
2. [ ] 현재 학급 개수 확인
3. [ ] 임의의 학급 카드에 마우스 호버 → 삭제 버튼(🗑️) 표시 확인
4. [ ] 삭제 버튼 클릭
5. [ ] 확인 창에서 "확인" 클릭

**예상 결과:**
- [ ] ✅ 삭제된 학급이 **즉시 사라짐**
- [ ] ✅ 학급 개수 및 학생 수 통계 즉시 업데이트
- [ ] ✅ 새로고침 없이 실시간 반영

**실패 조건:**
- [ ] ❌ 모달을 닫고 다시 열어야만 삭제가 반영됨

---

### 테스트 3️⃣: 종목 선택 시 자동 모달 오픈 (개인 종목)

**목적:** 개인 종목 체크박스 클릭 시 출전 선수 선택 모달 자동 오픈

**단계:**
1. [ ] "경기 종목 선정" 탭 클릭
2. [ ] "개인" 탭 선택
3. [ ] 임의의 개인 종목 체크박스 클릭 (예: "30초 달리기")

**예상 결과:**
- [ ] ✅ 체크박스가 선택되면서 **즉시** 파란색 모달창 자동 오픈
- [ ] ✅ 모달 제목: "{종목명} - 출전 학생 선택"
- [ ] ✅ 모든 학급이 목록에 표시됨
- [ ] ✅ 각 학급 기본 펼쳐진 상태

**실패 조건:**
- [ ] ❌ 체크만 되고 모달이 뜨지 않음
- [ ] ❌ 모달이 뜨지만 학급이 표시되지 않음

---

### 테스트 4️⃣: 종목 선택 시 자동 모달 오픈 (짝 종목)

**목적:** 짝 종목 체크박스 클릭 시 팀 구성 모달 자동 오픈

**단계:**
1. [ ] "경기 종목 선정" 탭에서 "짝" 탭 선택
2. [ ] 임의의 짝 종목 체크박스 클릭 (예: "짝줄넘기")

**예상 결과:**
- [ ] ✅ 체크박스가 선택되면서 **즉시** 초록색 모달창 자동 오픈
- [ ] ✅ 모달 제목: "{종목명} - 팀 구성"
- [ ] ✅ "짝 종목 (2명)" 표시
- [ ] ✅ 안내 문구: "학생 2명을 선택하면 자동으로 팀이 생성됩니다."

**실패 조건:**
- [ ] ❌ 모달이 뜨지 않음
- [ ] ❌ 개인 종목 모달이 뜸 (잘못된 모달)

---

### 테스트 5️⃣: 종목 선택 시 자동 모달 오픈 (단체 종목)

**목적:** 단체 종목 체크박스 클릭 시 팀 구성 모달 자동 오픈

**단계:**
1. [ ] "경기 종목 선정" 탭에서 "단체" 탭 선택
2. [ ] 임의의 단체 종목 체크박스 클릭 (예: "긴줄넘기")

**예상 결과:**
- [ ] ✅ 체크박스가 선택되면서 **즉시** 보라색 모달창 자동 오픈
- [ ] ✅ 모달 제목: "{종목명} - 팀 구성"
- [ ] ✅ "단체 종목" 표시
- [ ] ✅ 안내 문구: "원하는 인원을 선택한 후 '팀 생성' 버튼을 눌러주세요."

**실패 조건:**
- [ ] ❌ 모달이 뜨지 않음

---

### 테스트 6️⃣: 학급 없을 때 경고 메시지

**목적:** 학급이 없는 상태에서 종목 선택 시 안내 메시지 표시

**단계:**
1. [ ] 모든 학급 삭제
2. [ ] "경기 종목 선정" 탭에서 아무 종목이나 체크

**예상 결과:**
- [ ] ✅ 경고창: "먼저 학급을 등록해주세요."
- [ ] ✅ 모달이 뜨지 않음

---

### 테스트 7️⃣: 개인 종목 - 전체 선택 버튼 (학급별)

**목적:** 각 학급별 전체 선택/해제 버튼 작동 확인

**단계:**
1. [ ] 개인 종목 모달 오픈 (테스트 3 참고)
2. [ ] 첫 번째 학급의 **"전체 선택"** 버튼 클릭
3. [ ] 선택 상태 확인
4. [ ] **"전체 해제"** 버튼으로 변경되었는지 확인
5. [ ] "전체 해제" 버튼 다시 클릭

**예상 결과:**
- [ ] ✅ "전체 선택" 클릭 시:
  - 해당 학급의 모든 학생 버튼이 파란색으로 변경
  - 체크 아이콘 표시
  - "X명 선택" 카운터 업데이트
  - 버튼 텍스트가 "전체 해제"로 변경
  - 버튼 색상이 빨간색으로 변경
- [ ] ✅ "전체 해제" 클릭 시:
  - 모든 학생 버튼이 흰색으로 변경
  - "0명 선택" 표시
  - 버튼 텍스트가 "전체 선택"으로 변경
  - 버튼 색상이 파란색으로 변경

**실패 조건:**
- [ ] ❌ 일부 학생만 선택됨
- [ ] ❌ 버튼 클릭 시 반응 없음

---

### 테스트 8️⃣: 개인 종목 - 부분 선택 상태에서 전체 선택

**목적:** 일부 학생이 이미 선택된 상태에서 전체 선택 동작 확인

**단계:**
1. [ ] 개인 종목 모달 오픈
2. [ ] 한 학급에서 학생 2-3명 수동 선택
3. [ ] 같은 학급의 "전체 선택" 버튼 클릭

**예상 결과:**
- [ ] ✅ 기존 선택된 학생 포함하여 **모든 학생** 선택됨
- [ ] ✅ 버튼이 "전체 해제"로 변경

---

### 테스트 9️⃣: 짝 종목 - 전체 선택 버튼

**목적:** 짝 종목 모달에서 전체 선택 버튼 작동 확인

**단계:**
1. [ ] 짝 종목 모달 오픈 (테스트 4 참고)
2. [ ] 한 학급 펼치기
3. [ ] "학생 선택" 영역 옆의 **"전체 선택"** 버튼 클릭

**예상 결과:**
- [ ] ✅ 모든 학생이 초록색으로 선택됨
- [ ] ✅ 2명씩 자동으로 팀이 생성됨 (짝 종목 특성)
- [ ] ✅ 버튼이 빨간색 "전체 해제"로 변경
- [ ] ✅ 팀이 학급 헤더 우측에 인라인으로 표시됨

**실패 조건:**
- [ ] ❌ 전체 선택되지 않음
- [ ] ❌ 팀이 자동 생성되지 않음

---

### 테스트 🔟: 단체 종목 - 전체 선택 버튼

**목적:** 단체 종목 모달에서 전체 선택 버튼 작동 확인

**단계:**
1. [ ] 단체 종목 모달 오픈 (테스트 5 참고)
2. [ ] 한 학급 펼치기
3. [ ] "학생 선택" 영역 옆의 **"전체 선택"** 버튼 클릭
4. [ ] 선택 상태 확인
5. [ ] "팀 생성 (X명)" 버튼 클릭

**예상 결과:**
- [ ] ✅ 모든 학생이 보라색으로 선택됨
- [ ] ✅ "팀 생성 (X명)" 버튼이 활성화되어 보임
- [ ] ✅ 버튼 클릭 시 팀이 생성됨
- [ ] ✅ 전체 선택 버튼이 "전체 해제"로 변경

---

### 테스트 1️⃣1️⃣: 여러 학급에서 전체 선택 동시 사용

**목적:** 여러 학급의 전체 선택 버튼이 독립적으로 작동하는지 확인

**단계:**
1. [ ] 개인 종목 모달 오픈
2. [ ] 첫 번째 학급: "전체 선택" 클릭
3. [ ] 두 번째 학급: "전체 선택" 클릭
4. [ ] 세 번째 학급: 2명만 수동 선택
5. [ ] "확인" 버튼 클릭

**예상 결과:**
- [ ] ✅ 1반: 전체 학생 선택됨
- [ ] ✅ 2반: 전체 학생 선택됨
- [ ] ✅ 3반: 2명만 선택됨
- [ ] ✅ 전체 카운터 정확히 표시 (예: "총 47명 선택")

---

### 테스트 1️⃣2️⃣: 종목 해제 시 모달 미오픈

**목적:** 이미 선택된 종목을 해제할 때는 모달이 뜨지 않는지 확인

**단계:**
1. [ ] 이미 선택된 종목의 체크박스 다시 클릭 (해제)

**예상 결과:**
- [ ] ✅ 체크박스만 해제됨
- [ ] ✅ 모달이 **뜨지 않음**
- [ ] ✅ 선택된 종목 목록에서 제거됨

---

### 테스트 1️⃣3️⃣: 모달 저장 후 데이터 유지

**목적:** 출전 선수를 선택하고 저장한 후 다시 열었을 때 데이터 유지 확인

**단계:**
1. [ ] 개인 종목 모달에서 일부 학생 선택
2. [ ] "확인" 버튼 클릭하여 저장
3. [ ] 상단 선택된 종목 바에서 **같은 종목 카드 클릭**하여 모달 재오픈

**예상 결과:**
- [ ] ✅ 이전에 선택했던 학생들이 **그대로 유지**되어 표시됨
- [ ] ✅ 선택 카운터 정확히 표시
